# decision making and recursion

## what i did

i solved 2 problems using if else:
- leap year checker
- ticket pricing

and 2 using recursion:
- fibonacci
- palindrome checker

## idea

leap year:
year divisible by 4 but not 100 unless also 400

ticket:
based on age range

fibonacci:
each number = sum of previous two

palindrome:
check first and last character then repeat

## conclusion

decision making is simple using if else
recursion is useful but can be slow sometimes
